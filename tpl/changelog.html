Bootstrap4 更新日志

v4.3 系列主要更新了这些内容：

新增：添加了 .stretched-link工具，使任何锚点的大小与其最近的位置相对：相对父级，非常适合完全可点击的卡片！

新增：添加了.text-breakk工具，应用于word-break:break-wordk

新增：为 border-radius 添加了 .rounded-sm和 .rounded-lg。

新增：添加了响应式 .list-group-horizo​​ntal 修饰符类，用于将列表组显示为水平行。

改进：通过对变量使用 null 来减少我们编译的CSS，默认情况下从其他元素继承它们的值（例如， $ headings-color 是继承的，现在为 null，直到你在自定义 CSS 中修改它为止）。

改进：徽章焦点样式现在与按钮背景颜色匹配。

修复：在我们的 JS 插件中为 href HTML 属性消除了错误的选择器，以避免 JavaScript 错误。请尝试使用有效的选择器或数据目标 HTML 属性/目标选项（如果可用）。

修复：v4.3.1 恢复了对断点和栅格容器的更改， Sass 映射在修改这些默认变量时阻止人们升级。

修复：撤销修改：将 white-space: nowrap 应用到 .dropdown-toggle（在v4.3.1之前，它应用在所有 .btns 上）所以插入符号不会换行到新行。

弃用：img-retina，invisible，float和size mixins现已弃用，将在v5中删除。
